《介紹5個on-chain data web》


要做量化交易，淨係用price data (OHLC）係遠遠唔夠嘅，因為過去價格並未能有效預測未來價格，而喺cryp土世界有樣野叫on-chain data，即係喺鏈上嘅交易資訊係會公開透明，例如:


- Transaction data: 接收同發放嘅 addresses, transferred amount, address所own 嘅value…


- Block data: miner fee and rewards


- Smart contract code


要收集呢d零散數據去做分析十分須時，所以有唔同data service providers 將on-chain data整合成metrics 提供比用家：


1. Glassnode


• 2017 年成立，提供區鏈數據同分析，有interactive dashboard 比用家去做數據可視化

• 分tier 1-3，免費用家只能access 基本on-chain data例如number of addresses，要access所有resolution 嘅on-chain data而且想用api send request 嘅話，就要成為tier 3 會員，須一次性繳付一年費用，盛惠9588 usd


2. CryptoQuant


• 提供主要4類data: on-chain data; Exchange data; Miner data; Network data


• 大部分低resolution data只要sign-up free account就可以喺dashboard 展示，但不能下載data或使用api


• 價格同glassnode 一樣


3. Dappradar


• 實時追蹤市面上9成decentralized application (dApps) 嘅users/ 交易數字，可以以fundamental角度黎睇一隻cryp土資產背後有冇實際營運數據支持


• 出左自己嘅管治token 去做Dappradar決策同投票影響公司嘅未來方向，$RADAR holder更可以access DappRadar Pro 去攞實時數據


4. DefiLlama


• Defi TVL (Total Value Locked) aggregator，有dashboard/ chart by protocol/ chain 去睇tvl，用數據量化defi嘅growth rate，最近更新增NFT專區


• 免費historical data，歷史tvl數據可以用webscraping下載到


5. Coinglass


• 前身bybt，有自己手機app，費用全免，有唔同CEX嘅data streams，例如OI, funding rate, liquidations等等，亦有內置market indicator數據


• 所有data 都可以以webscraping download到，甚至resolution 到1 minute都有，因為個網背後其實係

一個REST api


• 內有coinglass 唔同CEX funding rate嘅webscraping 教學，只限Patreon 會員
