《一圖找出價量異動》


我地炒股炒幣經常都想搵到異動股/幣，然後買入搏佢嘅動量可以持續下去，而如何量化「動量」？同埋何謂「異動」？就係今日嘅topic


首先動量如何量度？普遍大家都會認為係價格回報率(return)，或者價格波幅率(volatility)，而亦有人會計算成交量（volume）因為越多成交就等於越多人同意此刻嘅價格，所謂「量價齊升」普遍認為利好就係咁解


得知動量嘅量度因子後，又如何定義「異動」呢？首先唔同嘅幣會有唔同嘅beta，價格預期回報率(expected return）唔同，例如btc嘅平均日回報為1-2%而doge為3-5%，所以我地唔可以用一個hard code percentage 去代表異動，成交量亦然，market cap越高嘅幣或股成交量亦越高。除此之外，唔同時期嘅asset expected return都唔同，單單係今年年頭嘅btc同Q2嘅btc波幅都大相逕庭


要搵到「異動」，要首先搵到何謂「常態」，要解決以上兩個問題，我地可以用rolling z score解決，因為z score 用了平均值同標準差去將數據標準化，而rolling （即每一個datapoint lookback n 個datapoint）可以只將最近嘅數據納入計算，盡量避免market 因時間大幅改變嘅問題


Patreon 附上了尋找異動幣嘅python code，通過計算價格與成交量嘅z score得出(圖一)嘅散點圖(scatter plot) 呢個圖可以輕易標示出價量異動嘅幣，例子中嘅threshold 為z score> 2


延伸閱讀：識搵異動幣或股= 賺錢？呢個問題等於係問緊呢兩個量度因子係咪有自相關性(autocorrelation)，係咪t=0價格越高，t=1嘅價格都會越高？呢樣野就留比大家backtest
