《虛擬貨幣相關性》


虛擬貨幣(Cryptocurrency)其實好幾年前已經開始hit，2016年bitcoin由幾百usd升到可以買到一部iphone嘅價錢嘅時候，吸引左普遍人注目，直到2017大牛市上到兩萬usd，再到今年升到六萬usd（相等於一架Tesla model X)，幣市開始進入白熱化階段，唔同幣商開始上市，好多新幣上架坐順風車，例如早兩日有elon musk撐腰嘅Shiba inu coin兩日升了600%，老散都開始紛紛入場


Cryptos嘅fundament呢度唔講，只講市場定價(market price)，無論劣幣良幣，幫你賺到錢嘅就係好幣，但係淨係講可以喺binance trade到嘅幣已經有千幾隻，眼花撩亂，應該買邊隻好？


要setup一個crypto portfolio，就先要有crypto價格嘅big picture，睇下唔同幣嘅相關性，因為即使portfolio 裡面有多種幣，若果幣與幣之間相關性高，都會令portfolio exposed to excessive risks


作為第一次講cryptos嘅intro，今次分享大家利用Python Seaborn package繪製虛擬貨幣相關系數熱能圖（Cryptocurrency Correlation Heatmap），比大家以鳥瞰角度觀察一下幣與幣之間嘅相關性（Correlation Coefficient 由-1至1，數值越大越相關，0為無關，負數為負相關性，heatmap圖中每格顏色深淺代表兩種唔同幣嘅相關系數，越紅越正數，越藍越負數）


從圖中可輕易見到BTC同ETH相關系數高（+0.79），即BTC升1%，ETH大概升0.79%，呢個事實可以應用喺pair trade之上，若然ETH升唔過0.79%，可以同時買入ETH和沽出BTC以賺取中間理論上差額，相反亦然


p.s. 為展示目的，數據來源自coinmetrics.io（daily data），若求更精準數據及higher frequency，請自行connect 幣商api, e.g. Binance api
