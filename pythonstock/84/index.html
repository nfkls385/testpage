雖然大家投票結果比較想睇用Python 做策略回測(backtesting)，但實際之前幾個post嘅反應黎睇比較多人想知點樣用Python 上網攞data，所以今星期再講一講點用Python 自動擷取即時恆指期貨數據


網絡抓取/爬蟲(Webscraping/Web crawling) 近年黎好流行，出面要上堂學都唔平，要幾千起跳，呢度免費分享比大家


首先大家要知道一般大家上網背後嘅流程，例如散戶A想上AASTOCKS睇騰訊分鐘圖表，背後電腦其實做緊：


1. 散戶Click入騰訊圖表網址(url)

2. 瀏覽器（e.g. Google Chrome)會向AASTOCKS伺服器（server）送出請求（request）

3. AASTOCKS 如果同意你嘅請求，就會就出回應（response），將網頁嘅原始碼（e.g. HTML JSON)傳比你嘅瀏覽器

4. 最後你嘅瀏覽器，將嗰堆原始嗎翻譯返成靚嘅圖表同文字（即係平時我地上網見到嘅樣）


所謂網絡爬蟲簡單講就係用程式扮係真人，直接向對方伺服器作出請求，從而截取佢嘅原始嗎，在從中抽取你想要嘅資訊


想自己寫到webscraping要有咩能力要求呢：

1. 基本Python language (普通slicing, pandas dataframe)

2. HTML 基本架構 (可以用package幫手，例如beautifulsoup)


當然，呢個只係最簡單嘅example，website架構複雜多變，每一個萬能嘅code可以拎到晒所有網嘅data，遲啲我都可以分享更加多嘅例子
