《介紹一個加密貨幣程式交易嘅Package》


相信做開程式交易嘅developer都同樣面對一個難題，就係每當轉換交易所嘅時候就要苦讀嗰個交易所嘅api document，例如今個月Binance唔再開放合約交易比香港虛擬貨幣投資者，大家紛紛轉會到二三線交易所，例如ftx(PI only), bybit 等等，要將本來喺Binance嘅algo script原封不動翻譯做其他exchange workable嘅script都有d困難，所以今日想介紹呢個一個打10個嘅training Library - CCXT


CCXT- CryptoCurrency eXchange Trading Library 係一個用於加密貨幣程式交易嘅JavaScript/ Python / PHP 庫，佢支援好多林林總總嘅虛幣交易所，大至Binance, ftx到bitmex, okex等等都係兼容，ccxt 提供快速訪問市場數據 market data 嘅途徑，例如簡單嘅ohlc historical data，orderbook, bid ask depth等等，或者data streaming/ websocket 都可以快速到，當然都支援大部份交易所以ccxt api 做order execution


我自己都有用開CCXT去儲存唔同exhange嘅數據，去搵arbitrage，開發程式交易策略同做backtesting，而且ccxt可以下載到嘅歷史數據比大部份交易所API容許下載嘅最長時期為長，例如deribit嘅quarterly/weekly contract 喺contract expired左之後就唔可以再喺deribit offical api download到，但ccxt就可以


Package 同source code喺github已經公開，python coding嘅話簡單一句pip install ccxt 就可以安裝到，快學快用
