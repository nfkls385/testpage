《散戶日記》港股強勢股突破策略賺到錢？
2020-06-07

最近有人問我港股炒股唔炒市，淨玩強勢股有冇得賺，即唔見平撈底，只買不斷創新高嘅股。因為佢見到即使個市大跌完整固緊嘅時候，好多股（e.g.物管/醫藥）已經升翻新高位置/創新高

我一向都跟海龜策略，覺得玩強勢股高追好過買一直下跌但覺得好平嘅股（例如5仔），所以試下做好簡化嘅backtest 睇下強勢股突破策略有冇得賺

1. 倉位只放10隻股，資金平均分配，每次換出1隻股都會補買另一隻股，總之倉位一直都有10隻股

2. 以市值/成交量篩選所有港股得出數百隻港股，每日睇每隻股前50日有冇創過200天新高，每創過200天新高一次果隻股就加1分

3. 每日收市前review portfolio，揀最高分嘅10隻股喺收市價買入，而跌出頭10嘅股就喺收市價賣出，而本來已持有頭10高分嘅股的話就hold住

4. 不設止賺止蝕，Long only strategy，不留現金，所有現金all in 股票，每日平均手續費0.2%，於Pnl扣除

Backtest 日子：2016/1/1-現在
-年化Sharpe: ~1.5
-Cumulative Pnl (compounded): 450%
-但standard deviation都有1.6%，波動過指數
-平均每10日換股1次，每次換平均1隻股，適合中短線投資者

另外parameter都可以改，如果倉位只想放5隻股，Cumulative Pnl可以到～600%，但std都會變大，反之，如果想分散風險，倉位可加至20-30隻，但 Pnl會下降

其實坊間有教我地呢類散戶每日開AAstock save低創新高股，日積月累就會得再一個強勢股list，但如果你識python/其他programming嘅話，呢d做功課嘅時間都可以慳返，可以放更多時間落實戰上
