今次分享嘅web crawling技巧比較advance，因為當中牽涉到破解對方網站反爬蟲(anti-scraping)工具，如果唔明白嘅可以靠trials and errors


反爬蟲手法有好多種，並唔係種種都能破解，例如Google 出嘅「請選擇所有含有紅綠燈的圖」就十分難破解，若果你破解到嘅話就可以去Google 做😂


而HKEX嘅反爬蟲手法唔太難，係用了cookies同token去阻止web scraper 用非正常途徑進入網站，免得佢哋Server不勝負荷。


而token同cookies會不時改變，而且係藏於主頁html code 入面嘅不明顯地方，要花時間喺主頁html code 入面search 段token嘅位置


而今次分享嘅分鐘data其實係HKEX用黎plot分鐘圖嘅data source黎，因為web developer喺寫網站嘅時候唔會咁蠢每隔一分鐘就改個網站，所以通常都會從第二個server/api 去feed data比個main website去更新，而只要我地搵到個server/api嘅link然後直接攞佢嘅data就可以（詳細請看link)
