《高夏普比率股票組合策略》
2020-06-24


1. 上個月整左個「強勢股突破策略」，暫時 forward test 個 sharpe 都有 3 ，個策略嘅 Logic 就係買入不斷創新高嘅股票，就好似眼前有 10 條 backtest 左嘅 strategy ，我地只揀 Equity Curve 不斷創新高嘅，而棄用不斷創新低嘅。


👇🏻👇🏻強勢股突破策略👇🏻👇🏻


2. 而單獨股票嘅回報本身都可以睇成一條 strategy ，例如 700 明顯就係一條好嘅 strategy 。


3. 要 evaluate 一條策略係咪高回報低風險，要睇 annualised sharpe ratio （夏普比率 )

{Mean of Daily Return/Standard Deviation of Daily Return * sq root (252) }

而統計上要有 at least 200 個 data point 即 look back 200 trading days 先有意思


4. 個策略邏輯好簡單：每日計算每隻股嘅 Sharpe ( 用 200 日 look back) ，買入或 Keep 最高 Sharpe 嘅頭 n 隻股，賣出跌出頭 n 高 Sharpe 嘅股，倉位永遠持有 n 隻股


5. 呢個策略嘅 sharpe 因 n 而變化，越多股 sharpe 越低，因為交易次數多， transaction cost 高，例如持有 10 隻股嘅 sharpe 有 1.8 ，只持有 5 隻可提升到 2.0 以上（ From 2016 to now)


6. 以上就係個策略嘅 logic ，而詳細 scoring 嘅 algorithm 就大家自己研究
