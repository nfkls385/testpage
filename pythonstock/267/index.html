港股update


1. 恆指2021年全球最差，全年-14%，由最高位下跌23%，過去恆指高位drawdown大約25%就會有大手掃貨反彈 ，由上年高位計算，低位大約22700

2. 恆指Z score 周期數字大約-1.6，大概-2左右就可以慢慢入貨（若不知Z score係咩可以睇翻我之前嘅文），呢d 位要去到-2可能要妹一段時間

3. 恆指PB<1 歷史黎講係相對低位，慢慢部署指數etf係唔錯嘅位

4. 歷史上嘅一月效應（一月通常中國政策公布，好政策有利股市）

5. 加息預期大增，對負債高嘅版塊（如部分科技股/地產等）不利，利好銀行股/保險股，長遠利好商品，匯豐5仔，恆生11等都喺相對高位


Crypto Update


1. Bitcoin由67000跌到琴晚41000，drawdown 40%，歷史drawdown可以到6x%，但因為derivatives market嘅流行，short side嘅出現會因為價格短期反彈而liquidated，進一步令價格有支持，所以要bitcoin跌翻落10000呢d位有難度

2. 個市10月尾出現大量槓桿，Open interest 大升，而12月頭果下大插雖然將槓桿油殺得7788，但不出一星期佢地急速加注，OI更加係破頂，usdt lending rate 急升，意味着好多人借貸做槓桿操作，個市唯有向下殺

3. 其實槓桿油急升嘅data早喺12月尾見到，但由於12月尾係大量衍生產品結算時間，更有70幾億48000 strike option 喺deribit 結算，所以市場一直妹到2022年頭先爆邊

4. 個市對於market neutral investment係unfavourable，basis（季度期貨同現貨差異）正常情況下係高水，代表投資者對後市睇好，今年十月basis apy更加高至30%，但呢家得大約5%，歷史回測黎講apy<=0%就係入市好時機

5. Greed and Fear index 大約10，歷史低於10就係買貨時機

6. 留意ONE, FTM, ATOM, 前兩隻TVL 破頂，代表大量資金流入，後者2022Q1公佈重大update，有機會開發跨鏈用戶
