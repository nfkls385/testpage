Backtesting 嘅最後一步就係做parameter optimisation，簡單黎講就係用唔同數值嘅parameter 去試條strategy 去達到我地嘅goal，通常我地嘅goal都係不外於Maximising Sharpe ratio, Maximising Return, Minimising Maximum Drawdown等等，而透過不停重覆以上步驟，我地會得到好多測試結果，再去搵set最好嘅parameter apply strategy 嘅trading rules


而有時parameter 可以多於一個，例如以上例子就係有兩個parameter: reference point 同threshold value, 為左更易睇到兩個parameter 對strategy 嘅影響，我地會用上heatmap，以顏色深淺黎輕易決定用邊set parameter ，而且如果見到heatmap 顏色分佈非常亂（parameter郁少少成績差多多）的話，條strategy 好大機會係overfit，成績好只係撞彩
