《點樣set python schedule task?》
2021-04-11


之前分享左好多爬蟲webscraping python script，包括HKEX下載option data, 南北水，恆指期貨，daily market report......等等，雖然已經寫咗段script十幾秒就可以自動download，而言要每日人手撳python個run掣都仲係有d煩，而且download漏幾日data就會令data整體可分析度減低


相信唔少人都知電腦可以set schedule task排程要做嘅task，例如一至五8:00am下載琴日嘅北水數據再用 email/tg send 交易signal比自己，但電腦default嘅schedule task tools係要你保持電腦喺非休眠狀態，而且需要上網下載data嘅task更加需要有網絡供應


如果想閂左部電腦都run到schedule job，其中一個最好嘅方法就係用Cloud Computer，上Google Cloud 申請Virtual Machine，基本上等於擁有一部永遠唔會閂機嘅電腦比你run schedule task，雖然佢係計你流量收費，但如果你只係用黎run 低流量嘅script （即係非machine learning)而且用非window嘅os，例如上網下載data，係完全唔會收費（本人每日run緊廿幾條script，一年來一蚊都冇收過）


而schedule job tools我會建議用crontab（通常用Linux OS)，除左可以處理例行性嘅task，仲可以記錄task log，睇翻run task 時候嘅所有資訊，而crontab 有自己嘅格式（* * * * * command）詳情可以睇返crontab wiki


最後想玩algo trading嘅朋友基本上最好學識點用cloud computer 黎run strategy，除非你租server去run
