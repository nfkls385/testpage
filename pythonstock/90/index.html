《如何用Python自動取得股票新聞》
2020-10-02


股票嘅新聞往往對股價有一定影響，雖然新聞較股價反應滯後，但新聞多少往往都能夠反映隻股票嘅熱度，係咪熱門股票等，亦都可以分析隻股票普遍係咪被睇好


散戶都多數睇etnet/aastock/富途嘅股票新聞，所以呢次教學都係用翻etnet做例子，自動去攞某股票嘅新聞內容


當然實戰上有冇用就睇大家點運用，有人會統計某股票新聞多少判斷該股票係咪市場熱門，從而篩選出熱門股票，利害d嘅可以做sentiment analysis，即係用Machine learning 方式判斷該新聞為正/反面嘅新聞，再統計一段時間嘅新聞，從而計算出該股票嘅分數（普遍睇好/唔好）


Webscraping make requests 基本上有兩種:

GET: simply fetching a webpage

POST: sending information to the server e.g. submit a form

而今次etnet 嘅例子係要填一張form先攞到股票新聞data，所以係用POST requests
