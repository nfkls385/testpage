傳統市場 vs Crypto市場（一）
2024-04-30

4月份小弟獲邀出席幾個不同場合，包括大學、證券行、電台、電視等等，去分享對加密貨幣市場的看法。除了上述活動外，亦被交易所邀請到泰國及杜拜出席活動及晚宴，因此沒有太多時間靜下來寫專欄。不過讀者們放心，一定會「追數」。


在不同講座及節目中，有一個極常被問及的問題，就是加密貨幣市場與傳統市場的分別。


我一直都有提過，之所以我經常交易加密貨幣，並不是我很愛加密貨幣，並不是我是一個Bitcoin maximalist，並不是因為我曾在幣市賺很多錢。而是因為加密貨幣市場相對無效（inefficient），相對其他市場容易賺錢。


如果幾年後，幣市已經變得有效，屆時出現另一個更無效的市場，我會果斷跳船。我是個trader，哪裡有錢，就去哪裡。


結果論一點說，我在傳統市場每年平均賺15-25%，sharpe ratio 2；但我在幣市年化過百，sharpe ratio近3，已經證明幣市風險更低，回報更高，因此更好賺。


最簡單原因是，機構還未完全進場，alpha仍然存在。


以上原因有密切追蹤我的朋友都會聽過。不過，這次專欄打算細講兩個市場的區別。


先講市場結構：


1. 交易所選擇


傳統市場交易所大多只有1-2個，例如港股只能在香港交易所買賣；日股只能在東京證券交易所及大版證券交易所買賣；新加坡股只能在SGX買賣等等。不過，加密貨幣交易所則百花齊放，包括Binance、OKX、Bybit、Coinbase等等。


導致幣市可以進行大量套利，包括cross exchange spot arb、cross exchange perp arb、cross exchange futures arb、cross exchange rate arb等等。


2. 市場開盤時間


傳統股票市場開始只是4-5小時，即使是國際期貨也是23小時，然後周一至周五。但幣市全部都是24小時、7天。因此，幣市並沒有gap up gap down，亦不會有所謂t+1、t+2結算交收等等。傳統市場因為有開市收市甚至有競價時段，因而有大量盤前盤後策略，亦有大量開收市才會執行的策略。相反，幣市並無分盤前盤後，亦沒有收市，能用的策略大不同。


不過，由於幣市會橫跨亞、歐、美時段，因此有人獨立設計出適用於亞或歐或美時段的策略。


經常有人跑來裝懂問我在幣市做的是intraday還是overnight策略，我大多忍不住笑。


3. API connectivity


傳統市場與交易所溝通用的大多是FIX protocol，而炒幣與交易所溝通用的大多是web based的，如REST API及web socket API。前者難度更高，大多要請有經驗的developer專寫FIX。而web based則相對容易，不過，延時也會較明顯。


傳統市場的機構投資者更會採用colocation，把服務器搬近交易所，甚至會租leased line、租microwave tower、租海底光纖，去減低延時。因此，傳統市場的高頻已經是硬件軍事競賽，絕對是大戶專利；相反，幣市並非如此，不同參與者都受到一定程度網速限制，大戶散戶差距相對不大。
