3者都在致力發展DeFi業務，衡量DeFi的其中一個指標為TVL，即Total value locked，可以想成，有多少錢在它們的生態圈中，就如Bitcoin的價格與Bitcoin的Transaction amount成正比。所以TVL愈高，該Blockchain愈值錢。


ETH TVL為114 B

BSC TVL為17.5 B

SOL TVL為3.1 B

這樣看來，ETH比BNB比SOL市值高是很合理的。


下一步可以把Market cap除TVL，以看看市值與Locked Value是否成比例，就像我們炒股看PE、PB一樣，數字愈高愈超買。


計算如下（8月31數據）：

ETH 383/114 = 3.4

BNB 80/17.5 = 4.6

SOL 31/3.2 = 10


看畢後，你可能驚訝SOL是多麼昂貴！等等，先別急著賣掉SOL，SOL PE高是很合理，因為SOL的fundamental比BSC比ETH強得多。重點來了，我怎知道SOL PE要高另外兩者多少才合理呢？


我拿來了3大數據：

transaction per second、

average fee per transaction、

transaction latency


把每一項指標與3者的PE作regression， 以看看每條blockchain的fundamental強弱是否與PE高低成正比。


請看圖2，你會驚訝市場的Pricing是如此的精準！


每張小圖的x-axis是fundamental數據，由左至右是transaction per sec、transaction fee、transaction latency，而y-axis則是mkt cap / TVL。3張小圖的R^2分別為0.997、0.978、0.999，即3者的3項基本面均與他們的PE非常貼合，市場計價分毫不差。


OK，即其實ETH、BNB及SOL都身在fair price，那是不是不用trade了？


非也，fundamental數據是固定的，但TVL每天都在變，愈多錢lock up，該幣就會升。用另一角度看mkt cap / TVL，即每$1可以貢獻該幣多少mkt cap：例如每$1進入eth defi，eth市值就升$3.4；每$1進入sol defi，sol市值就升$10。


因此，策略為，看著TVL，當TVL升跌，而幣價不按比例升跌，就是我們賺錢的機會。


還有另一個重點，若你細心留意一下TVL，是很momentum的，因此若TVL一直在上升，基本可以預期幣價也會節節上升。
