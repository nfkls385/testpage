投資要賺錢 看這一篇就夠
2023-01-16

我做投資者教育已經6年，前後寫了二百幾萬字專欄，幾乎涵蓋了所有大家賺錢需要知道的知識，例如金融學、統計學、編程、數學、心理學、博弈論、各種bias等等，但仍會有人走來inbox問，「聽日恒指睇升定跌？」、「Bitcoin依家買唔買得過？」、「我供公用股輸好多點算？」，仍會有人inbox說「你patreon再唔俾number我就unsubscribe」。


賺錢的方法、策略、心態已經寫了很多，這次patreon我寫一個我會對所有新手說的事情。


不時會有新朋友走來問「炒野點樣可以贏錢，點入手？」。答了6年的我，幾乎已經standarise了一個答案，就是：「你有無時間？會唔會抗拒數字？如果無時間，就DCA；如果有時間，又唔抗拒數字，就寫algo啦」。


這是我認為最簡潔而又幫到新手的最佳答案。因為對著抗拒數字的人推薦quant trade及algo trade只是對牛彈琴，但如果有時間，我120%肯定algo是全個二級市場最賺錢的炒賣方法。


因此，最高效率、最高性價比的方向只有兩個，要不最懶直接月供ETF或一些優質資產，就是研究quant trade及algo trade。


跟據過去6年投資者教育的經驗，以及跟據我16年的投資經驗，只有這兩種方式最能賺錢。可以看看附圖，x軸為知識及認知水平，即炒賣知識，包括金融、統計、數學等知識，而y軸則為投資回報。你會發現整個curve呈j shape，左則及右則為正回報，而中間部分則為負回報。


先討論最左部分，需要的知識接近0，或者說，是沒有多少炒賣知識的人。如果這些人投資，他們會做甚麼？當然就是傻傻地供MPF，或者買銀行/保險sales sell他們的基金。供的要不是美股（SPY），就是港股（2800），長年平均年回報為4-8%不等。有些risk averse的打工仔則會供債券基金，長年平均回報則為2-6%不等。總結，傻更更地月供資產普遍會有單位數的正回報，因為他們不懂，所以甚麼也不會做。


好，到中間部分，為稍有炒賣知識的人，這些大多數炒賣年資為1-5年，多數會懂簡單技術分析及基本分析，懂各種陰陽燭形態，懂各種圖表形態如收窄三角、頭肩頂、VCP等等，懂技術指標如Bband、RSI、MACD等等，懂甚麼是PE、PB、股息率等等。這些人10個有8個炒賣一、兩年後都是損手，這是千古不變定律，我開始炒賣的頭幾年也是。這班人虧損後，都會有著認為自己運氣不好的想法，並認為自己需要更努力地磨練，並認為只要調節好心態就能贏錢。


再看中間偏右部分，即認知水平比初哥高些，就是上述提及的初哥更認真學習後的階段。這階段沒有平均年資，較多取決於那個人的自信及操作方法。不少投資者都會有非常努力鑽研的階段，例如不斷閱讀財經專欄、不斷看youtube影片，更會到書局找不同的財經書藉看，尤其是暢銷書。因為學多了，所次會覺得自己認知水平已大幅提升，因而加大投入注碼及密度，開始嘗試即日鮮、嘗試找出爆升股、嘗試增加進出頻率。這些人不單懂基本技術指標及基本分析，更會學到些坊間普遍不會提及的特別方法。但就是因為投入注碼變大，以及密度增加，所以虧損幅度會比初投資時更大。雖然認知水平提升了，但平均回報極差，虧損超過5成一點也不罕見。


最後，為curve的最右方，當認知水平增加到某個臨界點，平均回報就會由虧轉盈。這個臨界點的觸發因素就是學對方法。對於我來說，我的這個點，就是學懂backtest。除了我自己的經歷外，我發現，當我的學生學懂backtest後，也同樣由虧轉盈，這更深化論證了backtest及quant trade是這個臨界點左右的分野。


學懂Backtest後，等於學懂驗證哪些投資方法能賺錢，哪些投資方法是虧損的。當你有能力分辨賺錢與不賺錢的方法後，還能再輸錢嗎？


於最右部分，能再賺多少就取決於策略種類及頻率。因為方法正確，即expected value為正數，所以absolute return多少就看n了。expected return x n = total return，很簡單的數學。普遍中頻策略一年回報為10-30%不等，一定比傻傻月供的人高，大概就是普遍對沖基金能做到的回報吧，例如Bridgewater、文藝復興、Millennium、2 Sigma、AQR等等。而把頻率提高後，就會變成高頻交易。市面上的market maker、HFT firm就是curve的這個部分，平均年回報由雙位數至三位數不等。沒有吹水，過往專欄已提及多次，這些公司的trader每年bonus大約為1-5年不等。


這patreon篇幅稍長，不過簡而言之，要賺錢，最高效的方法要不就是最懶，認為自己甚麼也不懂，傻傻的月供就可以；或者認真學習backtest、quant trade、algo trade，用最正確的方法炒賣；那些以為自己很懂，但其實不懂的階段最危險。


大家加油！
