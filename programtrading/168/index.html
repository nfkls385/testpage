10個坊間極錯的炒賣認知
2023-05-26

不少人常問為什麼我要做教育，因為眼見市場對金融的認知薄弱得很，概念也錯得交關，實在慘不忍睹，因此忍不住想糾正一下，以正視聽。就等於你發現同事或朋友寫錯字/讀錯字時，也會忍不住出手/出口糾正。這就是其中一個想做教育的原因。


以下將列出10個常見誤解/錯誤：


1. call佢，put佢


經常聽到股民或討論區的人說call佢、put佢，這實在是很難聽/很難入耳。Call及Put是工具，是期權，意義上是不能「期權恒指」、不能「期權騰訊」的。Call及Put就像ETF、窩輪、牛熊證、期貨，是工具。你會「明天打算期貨騰訊」、「昨天窩輪了巴巴」嗎？正確的說法是「long/buy佢」或「short/sell佢」；若想做期權，則是「long call / short call佢」或「long put / short put佢」。


每次一聽人說「call左佢」或「put左佢」就知道是初哥。


2. 期權不過夜不收時間值


幾年前，在一位教授期權的人的facebook上看到個極荒謬的post。那人出post寫「如果不想付期權的時間值，不持倉過夜就可以」。最可怕的是，like過百，comment幾十，紛紛「多謝呀sir」，可悲也。


最可悲的是，這位「導師」有開設期權課程，是誰大家自己找找。正確答案是，期權每分每秒也有time value decay，亦即theta。


3. 期指高水暗示大戶睇好後市


坊間不少所謂「導師」會留意期貨高低水，然後看到高水就指大戶看好後市，同學們要持貨云云。留意，期貨計價公式內，除了有股息外，亦有利率。因此，期貨是有時間值的！


利率愈高，時間值愈多，期貨愈應比現貨（指數）高。因此，若該月沒有派息，期貨是絕對應該高於現貨的！不要再說甚麼高水暗示大戶睇好後市，簡直極之荒謬。


4. 牛熊證無時間值


所謂「導師」常說窩輪才有時間值，而牛熊證沒有。這是錯的。牛熊證真正身分是barrier options，因此是種期權，只不過所有牛熊證都是深入價內barrier options。那怕有多深入，只要未過期，都會有時間值！


因此，不要再信「導師」胡亂教學的內容。


5. 你day trade定swing trade？


坊間股民愛問人day trade還是swing trade。這句說話已有毛病。Day trade是timeframe，是trading frequency維度的問題；而swing trade則是風格或策略。再者，專業的說法會問，是否intraday還是overnight策略，而非「是否day trade」。


另外，swing trade也非專業說法。行內如果做cta的style有兩類：momentum、reversion，中文前者為趨勢，後者為迴歸或反轉。


因此，日常交流中很容易能判斷對方是否行內人。
